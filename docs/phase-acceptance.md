# 阶段交付验收清单

## Phase 1：基础底座

- 工程结构符合前后端分离、多端独立、按板块分目录规范。
- 后端具备 RBAC 基础表、用户角色权限模型、操作日志表。
- 后端具备统一审批流基础表和状态枚举。
- 后端具备学生主档案、家校居家报备样板接口。
- 数据中台具备学生成长概览样板接口。
- PC 管理后台具备登录、布局、家校报备、学生档案、审批样板页。
- 小程序具备首页、居家报备样板页。

## Phase 2：教务与考评

- 完成学科、年级、班级、排课、成绩、作业、教学进度。
- 完成学生评优、教师评优、基础绩效指标。

## Phase 3：人事薪资

- 完成人事档案、入转调离、考勤、薪资项目、多级薪资审批、电子工资条。

## Phase 4：财务保险

- 完成收费、缴费、欠费、退费、对账、分账、保险投保、理赔。

## Phase 5：后勤心理安全

- 完成食堂、宿舍、校车、商超、物资、心理测评、风险预警、安全事件。

## Phase 6：招生升学

- 完成招生展示、意向登记、咨询跟进、升学规划、选科指导。

## Phase 7：数据驾驶舱

- 完成经营、教学、财务、安全、招生、绩效综合大屏。
