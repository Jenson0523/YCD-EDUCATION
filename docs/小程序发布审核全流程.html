① 前置准备（微信公众平台）

目标：完成小程序账号注册、认证、域名配置，确保具备发布条件。

步骤操作位置详细说明
1. 注册账号 mp.weixin.qq.com 选择企业主体注册（个人主体无法使用支付、附近小程序等能力）
2. 企业认证 设置 → 基本设置 → 微信认证 支付 300 元/年认证费，认证后可使用更多接口能力
3. 获取 AppID 开发 → 开发管理 → 开发设置 复制 AppID（小程序ID），填入 manifest.json
4. 配置服务器域名 开发 → 开发管理 → 开发设置 → 服务器域名 添加 https://node.alois.vip（request合法域名、uploadFile合法域名、downloadFile合法域名）
5. 配置业务域名 开发 → 开发管理 → 开发设置 → 业务域名 添加 node.alois.vip（web-view 可使用）
6. 完善基本信息 设置 → 基本设置 上传头像、填写名称、简介、选择服务类目（教育-在线教育）
💡 提示：服务器域名必须支持 HTTPS，且已备案（中国大陆）。当前 node.alois.vip 已配置 SSL 证书，符合要求。
⚠️ 注意：小程序名称一旦设置，修改需要重新审核。建议提前确定好名称。

③ 微信开发者工具（导入与调试）

目标：使用微信开发者工具导入项目，进行预览调试，确保无错误后再上传。

操作说明
下载开发者工具 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
导入项目 选择 dist/build/mp-weixin/ 目录，填入 AppID，项目名称填「云辰盾」
编译预览 点击「编译」按钮，模拟器显示首页即正常
真机调试 点击「真机调试」，用手机微信扫码，验证实际效果
体验版 上传后为体验版，可在公众平台添加体验成员，扫码体验
⚠️ 必查项：在开发者工具中，按 Ctrl+Shift+M（或点击「详情」→「本地设置」）确认：
• 不校验合法域名已关闭（上线前必须关闭）
• ES6 转 ES5 开启
• 上传代码时自动压缩开启

④ 上传代码（开发者工具）

目标：将构建好的代码上传到微信公众平台，生成开发版本。

操作流程：

  1. 在微信开发者工具中，点击顶部工具栏的 「上传」 按钮
  2. 填写 版本号：建议与 manifest.json 中 versionName 一致，如 0.1.0
  3. 填写 项目备注：描述本次更新内容，如「首次提交，包含请假、人脸核验、消息中心等功能」
  4. 点击 「上传」，等待上传完成
  5. 上传成功后，登录 微信公众平台 → 管理 → 版本管理，可以看到刚上传的开发版本
💡 版本号建议：遵循语义化版本规范（major.minor.patch），首次提交用 1.0.0，迭代用小版本号。

⑤ 提交审核（微信公众平台）

目标：填写审核信息，提交微信团队审核。

填写项云辰盾小程序建议填写内容
服务类目 教育 在线教育 或 教育-学校教育
需提供《办学许可证》或《事业单位法人证书》
标签 家校共育、请假管理、校园安全、人脸核验
功能页面 选择 pages/index/index（首页）
测试账号 提供一个可用的家长/教师测试账号（手机号+密码），方便审核人员登录体验
审核截图 上传小程序主要页面的截图（2-5 张），展示核心功能
隐私协议 需在 pages/index/index 或独立页面提供隐私保护指引
⚠️ 常见驳回原因：
1. 服务类目与实际功能不符 → 选择正确的教育类目
2. 没有提供测试账号 → 务必填写可登录的测试账号
3. 隐私协议不完整 → 在首页增加「隐私政策」入口
4. 个人主体提交了需要企业资质的功能 → 完成企业认证
5. 内容涉及用户生成内容（UGC）但未配置内容安全审核 → 接入微信内容安全 API

⑥ 微信审核（等待期）

目标：了解审核时间，处理可能的驳回。

审核阶段预计时间说明
机器审核几分钟自动检查代码包大小、敏感词、API 权限等
人工审核1-7 个工作日审核人员实际体验功能，判断是否符合规范
加急审核2 小时内每月有 3 次加急机会（需申请）

审核结果通知：

  • ✅ 审核通过：公众平台站内信 + 绑定的管理员微信通知，状态变为「审核通过，待发布」
  • ❌ 审核驳回：通知驳回原因，修改后重新提交审核（不占用新的审核配额）
💡 加快审核技巧：首次提交建议在周一至周四提交，避开周五和节假日，审核速度更快。

⑦ 正式发布（公众平台操作）

目标：审核通过后，将小程序发布到线上，用户可搜索访问。

发布操作：

  1. 登录 微信公众平台
  2. 进入 管理 → 版本管理
  3. 在「审核通过」的版本卡片上，点击 「发布」 按钮
  4. 选择发布方式：
    • 全量发布：所有用户立即使用新版本（推荐首次发布选择此方式）
    • 灰度发布：先对 5%/10%/20% 用户发布，观察无问题后全量（适合重要更新）
  5. 发布成功后出现 「线上版本」 卡片
✅ 发布成功标志：「线上版本」显示当前版本号和发布时间，用户已可搜索并访问。

⑧ 用户访问（推广与运营）

目标：让用户找到并使用小程序。

访问方式操作说明
微信搜索 微信顶部搜索栏输入「云辰盾」，点击小程序结果进入
扫码进入 在公众平台「推广 → 小程序码」下载小程序码，打印或分享
分享卡片 用户点击右上角「...」→「分享」，发送小程序卡片给微信好友
公众号关联 若有微信公众号，可在公众平台关联小程序，公众号菜单跳转
附近小程序 在公众平台开启「附近的小程序」，按地理位置展示

数据分析：

发布后，可在公众平台 统计 模块查看：访问人数、访问次数、留存率、页面访问深度等数据，指导后续迭代方向。
