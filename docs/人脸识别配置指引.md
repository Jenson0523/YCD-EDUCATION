## 🚀 三步开通百度智能云（推荐）

### Step 1：注册百度智能云并创建应用

1. 访问 [百度智能云控制台 - 人脸识别](https://console.bce.baidu.com/ai/#/ai/face/overview/index)
2. 登录（手机号注册即可，无需企业认证）
3. 点击左侧「人脸识别」→「概览」→ 点击「创建应用」
4. 应用名称：`云辰盾`，勾选 **「人脸识别」**
5. 接口选择：勾选 `人脸对比`、`人脸搜索`（按需）
6. 点击「立即创建」
7. 创建成功后，在「应用列表」中查看：
   - **API Key**：例如 `xxxxxxxxxxxxxxxxxxxxxxxx`
   - **Secret Key**：例如 `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

> 📸 截图指引：应用列表 → 点击应用名称 → 查看 API Key 和 Secret Key

### Step 2：配置环境变量

在服务器上设置环境变量后重启应用：

**Linux / macOS**：
```bash
export FACE_PROVIDER=baidu
export BAIDU_API_KEY=你的API_Key
export BAIDU_SECRET_KEY=你的Secret_Key

# 重启应用
java -jar ycd-api.jar
```

**Windows 服务器 (PowerShell)**：
```powershell
$env:FACE_PROVIDER = "baidu"
$env:BAIDU_API_KEY = "你的API_Key"
$env:BAIDU_SECRET_KEY = "你的Secret_Key"
```

**开发环境（IDE 启动）**：
在 IDEA 的 Run Configuration → Environment Variables 中添加：
```
FACE_PROVIDER=baidu
BAIDU_API_KEY=你的API_Key
BAIDU_SECRET_KEY=你的Secret_Key
```

> ⚠️ 密钥**绝对不能**提交到 Git！仅通过环境变量传递。

### Step 3：验证切换成功

重启应用后，访问：
```
GET http://localhost:8081/api/leave/face/engine-info
```

返回 `"engine": "baidu-cloud(百度智能云人脸识别)"` 且 `"ready": true`，表示开通成功！

## 其他引擎切换

```bash
export FACE_PROVIDER=tencent
export TENCENT_SECRET_ID=你的SecretId
```

```bash

# 无需任何密钥，重启即生效
```

```bash
export FACE_PROVIDER=arcsoft
export ARCSOFT_APP_ID=你的APP_ID
export ARCSOFT_SDK_KEY=你的SDK_KEY
```

## 常见问题

### Q: 我已经设了环境变量，但 engine-info 还是 image-hash？
**A:** 检查两点：
1. 环境变量是否在**启动应用前**设置的？（IDE 中需在 Run Configuration 设置）
2. 调用 `GET /api/leave/face/engine-info` 查看 `ready` 字段 — `false` 表示密钥为空或无效

### Q: 百度云返回 "error_code: 18" 什么意思？
**A:** QPS 超限（每秒超过2次请求）。前端加防抖/节流即可。

### Q: 百度云返回 "error_code: 222202" 什么意思？
**A:** 图片中未检测到人脸。请确保抓拍照片光线充足、人脸清晰可见。

### Q: 门卫刷脸总是通过不了？
**A:** 阈值设为 80，低于此分数判定为不匹配。建议：
- 录入时用正面清晰照片（无刘海遮挡、光线均匀）
- 门禁摄像头尽量正面拍摄
- 查看具体分数：PC后台 → 请假离校 → 请假台账 → 人脸通行台账

### Q: 如何查看详细的人脸比对日志？
**A:** PC后台 → 请假离校 → 请假台账 → 人脸通行台账，每次比对都有完整记录（分数、引擎、通过/拒绝）。
