# 安全与隐私设计

## 数据分级

- 公开：校园动态、招生展示内容。
- 内部：班级、课程、公告、普通考勤。
- 敏感：薪资、人事合同、家庭情况、心理记录、保险理赔、身份证号、手机号。
- 高敏：高危心理预警、重大事故记录、财务分账。

## 后端强制要求

- 每个接口校验登录态、角色权限、数据范围。
- 写操作记录操作日志。
- 高敏数据读取也记录审计日志。
- 敏感字段落库前加密。
- 日志输出前脱敏。
- 上传文件限制类型、大小、访问范围。

## 前端要求

- 前端隐藏菜单只作为体验优化，不作为安全边界。
- 高敏页面需要二次确认或水印。
- 导出文件必须经过后端权限校验。
