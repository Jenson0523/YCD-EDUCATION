# 标准化业务模板

模板统一维护，前后端导入、导出、打印、电子归档均以此目录为准。

## 分类

- `family-school`：居家状态报备表、家校共育记录表、沟通台账。
- `student`：学籍导入模板、成长档案导出模板、评优登记表。
- `academic`：教学计划、听课记录、成绩导入模板。
- `hr`：入职登记、离职交接、薪资项目、工资条。
- `finance`：收费项目、退费申请、对账单。
- `insurance`：投保清单、理赔材料清单、事故处置记录。
- `logistics`：食堂台账、校车排班、物资领用、商超库存。
- `psychology-safety`：心理访谈、风险评估、安全隐患整改。
- `enrollment`：意向登记、咨询跟进、选科建议。
