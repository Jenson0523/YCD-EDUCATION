# frontend/pc-admin

PC 管理后台，面向校长、股东、教务处、人事、财务、招生、行政后勤、班主任、心理安保校医等角色。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Pinia
- Vue Router

## 路由原则

每个业务板块对应一个路由分组：

- `/dashboard`
- `/family-school`
- `/student`
- `/academic`
- `/hr`
- `/finance`
- `/insurance`
- `/logistics`
- `/psychology-safety`
- `/enrollment`
- `/workflow`

## 开发

```bash
npm install
npm run dev
```
